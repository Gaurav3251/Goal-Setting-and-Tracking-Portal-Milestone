# Milestone — Goal Setting & Tracking Portal

Milestone is a role-based portal for quarterly goal planning and tracking:
- **Employees:** create goals and submit updates/check-ins
- **Managers:** review/approve goals and monitor progress
- **Admins:** manage cycles/users and view reporting/export insights

## Project link
- Live app link: https://goal-setting-and-tracking-portal-mi-psi.vercel.app/

## Tech stack
- **Frontend:** React 18 + Vite + TailwindCSS
- **Backend:** Node.js + Express
- **Database:** Supabase (PostgreSQL)
- **Auth:** Clerk
- **Email:** Resend
- **Charts/Exports:** Recharts, SheetJS
- **Hosting:** Vercel (frontend), Render (backend)
